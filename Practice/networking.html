Network Fundamentals

This section covers the absolute basics: what a network is, the different types you'll encounter, their physical and logical layouts (topologies), and the essential hardware that makes them work. Understanding these core concepts is the first step to mastering networking.

What is a Network?

A network is a collection of interconnected devices like computers, servers, and printers that can share resources and exchange data. Its purpose is to enable communication, share files and hardware, and centralize management.

Network Types

LAN (Local Area Network)

Covers a small area like a home or office. High speed, typically using Wi-Fi or Ethernet.

WAN (Wide Area Network)

Spans a large area, connecting multiple LANs. The Internet is the ultimate WAN.

PAN (Personal Area Network)

A small network for personal devices, like connecting a phone to headphones via Bluetooth.

MAN (Metropolitan Area Network)

Covers a city or large campus, bridging the gap between LAN and WAN.

Key Network Devices

Router

Connects different networks together and directs traffic between them using IP addresses. Your home router connects your LAN to the Internet (WAN).

Switch

Connects devices on the same local network (LAN). It's smarter than a hub, sending data only to the intended device using MAC addresses.

Modem

Converts signals from your Internet Service Provider (ISP) into a digital signal your router can use. It's the bridge to the internet.

Addressing & Protocols

For devices to communicate, they need unique addresses and common languages (protocols). This section explains IP addressing, the system that gives every device a unique identifier, and key protocols like DNS and DHCP that make the network function automatically.

IPv4 Addressing

An IPv4 address is a 32-bit number that uniquely identifies a device on a network. It's usually written as four numbers separated by periods.

192.168.1.10

A Subnet Mask (e.g., 255.255.255.0) is used to divide the IP address into two parts: the network ID and the host ID.

DNS (Domain Name System)

DNS is the internet's phonebook. It translates human-friendly domain names (like `www.google.com`) into computer-friendly IP addresses.

DHCP (Dynamic Host Configuration Protocol)

DHCP automatically assigns IP addresses to devices when they join a network. This avoids manual configuration and prevents duplicate IP addresses.

Security & Tools

Protecting a network is as important as building it. This section introduces fundamental security concepts and the command-line tools used by professionals to diagnose and troubleshoot network problems. Use the mock terminal below to see what these commands do.

Firewall

A security barrier that monitors and controls incoming and outgoing network traffic based on predetermined security rules. It's the first line of defense.

VPN (Virtual Private Network)

Creates a secure, encrypted "tunnel" over the public internet, protecting your data from being intercepted and providing privacy.
